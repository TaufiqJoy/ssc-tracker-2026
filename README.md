# SSC Tracker

A high-performance, glassmorphic countdown tracker for the SSC Examination 2026.

## 🚀 Features

- **Real-time Countdown**: Track days, hours, minutes, and seconds until the exam.
- **Glassmorphism UI**: Modern and premium design with smooth animations and gradients.
- **Responsive Design**: Works perfectly on desktops, tablets, and mobile devices.
- **Motivational Section**: Dynamic motivational messages to keep you inspired.
- **Auto-Completion**: Displays a "Best of Luck" message once the countdown reaches zero.

## 🌐 Hosting

This project is ready to be hosted on **GitHub Pages**. Simply upload the files to a repository and enable "GitHub Pages" in the repository settings.

## 📁 Files to Upload to GitHub

To host this website on GitHub, you need to upload the following files:

1. `index.html` (The main website structure)
2. `style.css` (The website styling)
3. `script.js` (The interactive features)
4. `README.md` (Project documentation)
5. `LICENSE` (Usage license)

## 🛠️ Usage

Simply open `index.html` in any modern web browser to view the tracker.
